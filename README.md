# Sql-data-warehouse-project
building a modern data warehouse with SQL , including ETL, data modelling and analytics


Developed an end-to-end SQL-based data pipeline to integrate, clean, and transform manufacturing and supply chain data from CRM and ERP sources into an analytics-ready data warehouse.

-Designed a Bronze → Silver → Gold architecture for scalable data processing and traceability

-Implemented advanced SQL techniques including window functions (ROW_NUMBER, LEAD), joins, and CTEs for deduplication, data integration, and temporal logic

-Performed robust data validation using NULL handling (NULLIF, COALESCE), conditional transformations (CASE), and business rule checks (e.g., pricing consistency, date integrity)

-Built dimensional models (Star Schema) with surrogate keys for customer and product analytics

-Integrated multi-source data and resolved inconsistencies (e.g., gender, country standardization)

-Enabled analysis of key manufacturing and supply chain metrics such as sales performance, product lifecycle, and customer behavior
